# Mini Microservices

## Project Setup

### Client

    $ npx create-react-app client

### Posts

    $ mkdir posts
    $ cd posts
    $ npm init -y
    $ npm install express cors axios nodemon

    $ npm install typescript ts-node
    $ tsc --init
    $ vim tsconfig.json

    $ npm install --save-dev @types/express

    $ npm run dev

### Comments

    $ mkdir comments
    $ cd comments
    $ npm init -y
    $ npm install express cors axios nodemon
